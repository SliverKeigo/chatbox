# ChatBox

一个基于 Tauri + React 开发的跨平台聊天应用，支持多种 AI 模型。

## 功能特性

- 🚀 基于 Tauri 2.0，性能优异
- 💬 支持多轮对话
- 📝 完整的 Markdown 支持
- 🎨 代码高亮显示
- 📋 代码块一键复制
- 💾 对话历史记录
- 🎯 支持自定义 API 端点
- 🌈 优雅的 UI 界面

## 技术栈

- Tauri 2.0
- React 18
- TypeScript
- OpenAI SDK
- React Markdown
- Syntax Highlighter

## 开发

### 环境要求

- Node.js 18+
- Rust 1.70+
- 系统要求参考 [Tauri 先决条件](https://tauri.app/v1/guides/getting-started/prerequisites)

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run tauri dev
```

### 构建应用

```bash
npm run tauri build
```
